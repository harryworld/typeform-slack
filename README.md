# typeform-slack

This is a little tool to automate the process of Slack Invitation

1. Create a typeform
2. Get the `formId` and `apiKey`
3. Create a Slack group
4. Create the `accessToken` for the Slack group
5. Fill in the `config` variable accordingly
6. Done!!
